# 우아한테크코스 준비 스터디 ✨

## 📌 스터디 목적
- 우아한테크코스 기출 문제 풀이 & 코드 리뷰
- 학습 회고를 통한 기록과 성장
- 자소서 작성 및 첨삭
- 프리코스 대비 및 최종 코딩 테스트 준비

---

## 🗓️ 스터디 일정

### 1주차 (9/16 ~ 9/22)
- 7기 1주차 과제 수행
- 상호 코드 리뷰 및 리팩토링
- 자소서 작성

### 2주차 (9/23 ~ 9/29)
- 7기 2주차 과제 수행
- 상호 코드 리뷰 및 리팩토링
- 자소서 상호 첨삭 및 수정

### 3주차 (9/30 ~ 10/6)
- 7기 3주차 과제 수행
- 상호 코드 리뷰 및 리팩토링
- 자소서 상호 첨삭 및 수정

### 4주차 (10/7 ~ 10/13)
- 7기 4주차 과제 수행
- 상호 코드 리뷰 및 리팩토링
- 자소서 상호 첨삭 및 마무리 (10/10 서류 제출 마감)

### 5주차~ (10/14 ~ )
- 프리코스 진행
- 프리코스 완료 후 최종 코딩 테스트 대비 기출 풀이

---

## 📖 스터디 진행 방식

### 매주 금요일까지
1. 각자 N주차 문제 풀이
2. 회고 보고서 작성  
   - 간단한 풀이 방식  
   - 참고한 자료  
   - 공유하고 싶은 것들  
   - (추가 항목은 점차 확장)

3. 깃허브 업로드

### 토요일 저녁까지
- 서로의 코드와 회고 보고서를 읽고  
- 질문/개선점/피드백을 PR로 남기기

### 일요일 (대면 모임)
1. 각자 코드 & 회고 발표 (리마인드 형식)  
2. 받은 피드백(PR)에 대한 답변  
3. 필요시 추가 토론 → Discussion 업로드
